
# SearchLit

A nodejs/express/jade(pug)/mongodb application to classify literary excerpts by genre using NLP classificaiton.

## Requirements

* [NodeJs](http://nodejs.org) >= 6.x 
* [mongodb](http://mongodb.org)

## Install

```sh
$ npm install
```

```sh
cp .env.example .env
```

```sh
$ npm start
```

[http://localhost:3000/](http://localhost:3000/)

## Run tests

```sh
$ npm test
```
